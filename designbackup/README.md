# designbackup

This folder is a GitHub-synced backup for the independent website.

## Contents

- `index.html`: backup copy of the current standalone site.

## Sync

After changes are committed and pushed from one computer, run this on the other computer:

```bash
git pull origin main
```

To refresh this backup after editing the live site, copy the latest root `index.html` into this folder, then commit and push again.
